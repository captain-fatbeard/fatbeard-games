# Flappy bird game

Code template for a flappy bird game


## Tech Stack

Vite, React, Typescript


## Run Locally

Clone the project

```bash
git clone captain-fatbeard/flappy-bird.git
```

Go to the project directory

```bash
cd flappy-bird
```

Install dependencies

```bash
npm install
```

Start the server

```bash
npm run dev
```
